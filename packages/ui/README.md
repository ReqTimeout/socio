# packages/ui — Design system shared

shadcn-svelte + Tailwind v4 (CSS-first `@theme`). Shared antara landing (Astro) dan app (SvelteKit).

## Tokens

- Color: light/dark (CSS variables)
- Spacing, typography, radius, shadow
- Status colors (7 warna order: Pending=amber, Proses=blue, Berhasil=green, Partial=purple, Batal=red, Error=rose, lain=slate)
- Brand: socio.id palette (port dari haloka — adaptasi)

## Komponen

Button, Card, Badge, Sheet (bottom-sheet), Dialog, Toast, Command, Input, Select, Tabs, Skeleton, Tooltip, Avatar, Accordion.
